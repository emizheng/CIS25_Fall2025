This is a program for the Diagnostics Test Tracker application written in C++ The purpose of this application is to store, track, and perform analysis on patient data and diagnostic test result data 
In order to run this application, be sure to download all contents of this Final_Project branch - everything here is needed to run the program except for the slideshow pdf file. 
Run the program by running the main.cpp file Instructions for how the program works is in the slideshow pdf file in this Final_Project branch
You may need to set up the sqlite3 connection using the sqlite3.c file before running the program - follow the first 7.5 minutes of this tutorial on youtube: https://youtu.be/L-hnA82JsEM?si=LsFedOH_sR0cNAFI
